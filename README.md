## Built With

React + Vite + React Bootstrap.

NodeJS + Express + Mocha + Chai.

## Getting Started

### Installation

1. Clone the repo
   ```sh
     git clone https://github.com/maoroz/asap-challenge.git
   ```
2. Install NPM packages
   ```sh
     cd asap-challenge
     cd server && npm i && npm run start
     cd client && npm i && npm run dev
   ```

## To Test 

### Server side:
```sh
npm run test
```
